About GandrLu

Hey, my name is Luzius and I create games, VR and XR experiences and web stuff under the label Gandrlu. Welcome to my website, you can find some of my projects here.

I create games and experiences mainly with Unity, my focus is on coding and gameplay design. Working on art (models, textures, sprites) and other assets are not unknown to me. I also pick it up from time to time and like it, but to save time and to be able to focus more on the other parts, I like to use external assets.

Developing experiences for the web, e.g. with A-Frame and THREE.js for 3D content, is another passion of mine.

Technologies and applications I work with:

  • C#
  • C++
  • Unity 3D
  • JavaScript
  • A-Frame
  • Qt
  • THREE.js
  • WPF
  • Blender
  • GIMP
  • Inkscape
  • Lens Studio

I like to participate in interesting projects to improve my skills further and learn new things. So if you stumbled somehow about this and think I could fit your project, send me a message: luzius_k[at]live.de

Cooling Module Game

This is a mini game that represents the cooling system of a PC. It is part of a bigger system with other computer components as (multiplayer) mini games, but this video shows only the fuctionality of the cooling system isolated, which was my part. The game and all assets are done by me (with 3DS Max and Unity) except:

  • Player (Model, Anims, Movement)

BearPong - MiniGameJam Berlin 11/2019

We made this little game where you take bears to the moon. Thanks to my teammates for this fun little project. Press the left mousebutton to charge a catapult and release it to shoot. Levels are randomly generated, when too hard, reload the page. Reach the moon!
